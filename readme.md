## Vuad
Example Atomic Design with DDD (Domain Driven Design)

## Requirement
  - [node.js](http://nodejs.org/)
  - [yarn](https://yarnpkg.com/en/)

## Quick Start

```bash
# clone repository
$ git clone https://github.com/logustra/vuad.git

# open folder vuad
$ cd vuad

# instal dependencies
$ yarn install

# copy file .env.example to .env
$ cp .env.example .env

# build and serve with express
$ yarn start
```
